Mas Vida Mas Futuro

Aplicación creada para ser usada como un blog ambientalista. Se pueden generar varios usuarios con diferentes permisos, escribir, editar y borrar artículos y comentarios. También posee un filtro de búsqueda avanzada de artículos por título, fecha, categoría y comentarios. Fue desarrollada en Django con una base de datos PostgreSQL y estilizada haciendo uso de Bootstrap.

Administrador de empleados

Aplicación creada con el propósito de gestionar empleados de una empresa. Se pueden añadir, editar y eliminar empleados. También posee una barra de búsqueda, donde se podrán buscar empleados por nombre, puesto, teléfono, etc. El Backend consta de una API Rest hecha en Spring-boot con base de datos MySQL y el Frontend fue diseñado en Angular.
